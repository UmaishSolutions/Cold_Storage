# Security Policy

## Reporting a Vulnerability

Do not open public GitHub issues for security reports.

Please report vulnerabilities to:

- Email: `solutions@umaish.com`
- Subject: `Security Report - Cold Storage App`

Include:

- Affected version/commit
- Reproduction steps
- Impact assessment
- Any proof-of-concept (if available)

We will acknowledge receipt and triage the report as quickly as possible.
